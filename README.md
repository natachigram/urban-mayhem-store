# Urban Mayhem Store

A decentralized Web3 gaming marketplace powered by blockchain technology and community trust.

## What It Does

Urban Mayhem Store is a trust-based item marketplace where players can:

- **Browse & Purchase Items** - Skins, weapons, bundles, and powerups for Urban Mayhem game
- **Blockchain Payments** - Pay with $TRUST tokens via secure on-chain transactions
- **Community Attestations** - Rate and vouch for items using Intuition Protocol
- **Trust Scores** - Item rankings influenced by community feedback and attestations
- **Purchase History** - Track all your completed transactions on-chain
- **UMP Currency** - In-game currency (Urban Mayhem Points) for purchases
